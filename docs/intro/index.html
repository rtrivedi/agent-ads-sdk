Welcome to AttentionMarket - the advertising exchange for AI agents and chatbots. Monetize your AI application by showing contextually relevant ads to your users.

Quick Exampleâ€‹

import { AttentionMarketClient } from '@the_ro_show/agent-ads-sdk';

const client = new AttentionMarketClient({
  apiKey: 'am_test_YOUR_KEY',
  agentId: 'agt_YOUR_AGENT'
});

// Get contextual ad based on user message
const ad = await client.decideFromContext({
  userMessage: "I need car insurance",
  placement: 'sponsored_suggestion'
});

if (ad) {
  console.log(ad.creative.title);  // "Save 20% on car insurance"
  console.log(ad.click_url);       // Auto-tracked click URL
  console.log(ad.payout);          // 250 (cents)
}

How It Worksâ€‹

  1. User interacts with your AI application
  2. You send context to AttentionMarket
  3. We match intent with relevant advertisers
  4. You display ad in your interface
  5. You earn money when users click
