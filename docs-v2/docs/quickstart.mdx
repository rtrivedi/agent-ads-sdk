# Quick Start

Get up and running with AttentionMarket in 5 minutes.

## Prerequisites

- Node.js 16+ (or Python 3.8+ for Python SDK)
- An API key from [api.attentionmarket.ai](https://api.attentionmarket.ai)
- An agent ID (provided during registration)

## Step 2: Get Your API Keys

1. Sign up at [api.attentionmarket.ai](https://api.attentionmarket.ai)
2. Create a new agent (or use an existing one)
3. Copy your API key and agent ID

You'll receive two types of keys:

- **Test keys** (`am_test_...`) — Use for development. Returns test ads, no charges.
- **Live keys** (`am_live_...`) — Use in production. Real advertisers, real revenue.

:::tip
Always use test keys during development to avoid affecting production metrics.
:::

## Complete Example

Here's a complete chatbot example:

```typescript
import { AttentionMarketClient } from '@the_ro_show/agent-ads-sdk';

const client = new AttentionMarketClient({
  apiKey: 'am_test_YOUR_KEY',
  agentId: 'agt_YOUR_AGENT_ID'
});

async function handleUserMessage(message: string) {
  // Get contextual ad
  const ad = await client.decideFromContext({
    userMessage: message,
    placement: 'sponsored_suggestion',
    minQualityScore: 0.7  // Only show high-quality ads
  });

  if (ad) {
    // Format ad for display
    const sponsoredMessage = {
      type: 'sponsored',
      disclosure: ad.disclosure.label,
      title: ad.creative.title,
      body: ad.creative.body,
      cta: ad.creative.cta,
      url: ad.click_url,
      payout: ad.payout / 100  // Convert cents to dollars
    };

    return sponsoredMessage;
  }

  return null;  // No ad available
}

// Usage
const ad = await handleUserMessage("I need car insurance");
if (ad) {
  console.log(`Showing ad: ${ad.title} (earn $${ad.payout} on click)`);
}
```

## Common Issues

### No ads returned

If `decideFromContext()` returns `null`, it means:

1. No ads matched the user query (try broader queries in test mode)
2. Your filters are too strict (remove `minQualityScore`, `minCPC`, etc.)
3. Your API key is invalid (check the dashboard)

### Clicks not tracked

Make sure you're using `ad.click_url` (not `ad._ad.url`). The `click_url` includes tracking parameters.

### TypeScript errors

Make sure to import types:

```typescript
import type { AdResponse, DecideRequest } from '@the_ro_show/agent-ads-sdk';
```
